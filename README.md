# Snake.py

Juego clásico de Snake realizado en Python utilizando Turtle.

# Modificaciones realizadas

Se realizaron las siguientes mejoras al código inicial:

- Se implementó movimiento aleatorio para la comida.
- La comida se mueve un paso a la vez.
- Se agregó validación para evitar que la comida salga de la ventana.
- Se modificó que cada que se inicie una partida la comida y la serpiente tengan colores randoms sin que tengan ambos el mismo y sin incluir el rojo.

# Controles disponibles

- Flecha derecha → mover a la derecha
- Flecha izquierda → mover a la izquierda
- Flecha arriba → mover hacia arriba
- Flecha abajo → mover hacia abajo

# Características del juego

- La serpiente crece al comer comida.
- El juego termina si la serpiente choca contra los bordes.
- El juego termina si la serpiente choca contra sí misma.
- La comida cambia de posición dinámicamente.

# Ejecución

Ejecutar el programa con:

python3 snake.py
